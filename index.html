PL-0 Compiler


This project is a PL-0 compiler with extened grammar written in C. It reads pl-0 source code file and generates program in assembly language and can run it. This compiler is also capable of reporting errors in source code file.

Skills Used: C, Compiler Theory


Below is an example. The PL-0 source code on the left is a simple program that reads a, b and compute power(a, b). On the right is the program in assembly language after compiling.

If the source code has errors, the compiler can detect where the errors are and report them with certain error#.
